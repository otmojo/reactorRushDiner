# 🚀 Reactor Rush Diner

Python による高パフォーマンス・レストラン・シミュレーター。
**マルチスレッド (Threading)** と **非同期 IO (Asyncio)** の圧倒的な性能差を可視化します。

## 5000 リクエスト

5000 件の同時リクエストにおける測定データ：

| モード | スループット (req/s) | CPU 仮想負荷 | 完了時間 |
| :--- | :--- | :--- | :--- |
| **Thread Pool** | **21.17 req/s** | 15.0% | ~240 秒 |
| **Async Core** | **8484.70 req/s** | 100.0% | **1 秒未満** |

> **結論**: 非同期処理（Asyncio）は、大量の I/O 待ちが発生するシーンにおいて、従来のマルチスレッドより約 **400 倍** 高速です。

## 🛠️ 主な機能
* **性能シミュレーター**: 数千の同時注文をシミュレートし、進捗をリアルタイムで表示。
* **デュアル・サーバー**: 非同期サーバー (Port 8888) とマルチスレッド・サーバー (Port 8889) を搭載。
* **サイバーパンク UI**: ANSI カラーによる視覚的なターミナル表示。
